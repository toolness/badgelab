This is badge lab.

Step ONE: Create issuer metadata

{ 
  "name": "An Example Badge Issuer",
  "url": "https://example.org"
}

Step TWO: Find a PNG for your badge

If you don't have time to create or find a good PNG, just use spacecat.

Step THREE: Define a criteria for your badge

Badge indicates person is cool.

Step FOUR: Create badge class metadata

{
  "name": "Awesome Robotics Badge",
  "description": "For doing awesome things with robots that people think is pretty great.",
  "image": "/spacecat.png",
  "criteria": "https://example.org/robotics-badge.html",
  "issuer": "https://example.org/organization.json"
}

Step FIVE: Create badge instance metadata

{ 
  "uid": "f2c20",
  "recipient": {
    "type": "email",
    "hashed": false,
    "identity": "foo@bar.org"
  },
  "issuedOn": 1359217910,
  "badge": "https://example.org/robotics-badge.json",
  "verify": {
    "type": "hosted",
    "url": "https://example.org/beths-robotics-badge.json"
  }
}

Step SIX: Bake the badge and issue it
